## ğŸš€ Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher installed
- A text editor or IDE (IntelliJ IDEA, Eclipse, VS Code, etc.)
- Git installed on your machine

### Installation

1. Clone this repository:
```bash
git clone https://github.com/Yoge-2004/JavaTutorials.git
cd JavaTutorials
```

2. Navigate to the tutorial or example you want to explore:
```bash
cd path/to/tutorial
```

3. Compile and run the Java files:
```bash
javac FileName.java
java FileName
```
